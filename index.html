Questions

Is this really 100% local? +

Yes. All file processing happens in your browser tab. When you connect Notion or Obsidian, data goes from your browser directly to that service — Luna has no intermediary server. We have no servers at all. Check the source on GitHub if you want proof.

Do I need WebGPU? +

Only for the AI features (summary, auto-tagging). Chrome 113+ on most modern computers has WebGPU by default. If you don't have it, non-AI features work normally and AI features fall back to a keyword heuristic.

Why $39 once instead of a subscription? +

Because these tools don't have any ongoing server cost for us — they run on your device. Subscriptions would be for something we aren't doing. We'd rather you pay us once and never see us again until you want an upgrade.

Can I use it for client work? +

Yes. One license covers commercial use on up to 3 devices. For team use (multiple people), contact us for team pricing.

Does it work on Firefox / Safari? +

Chromium-based browsers for now (Chrome, Edge, Brave, Arc, Opera). Firefox support is planned; Safari isn't on the roadmap because WebGPU support lags there.

What if I don't like it? +

30-day full refund, no questions asked. Email support@lunasuite.app and we'll send the refund the same day.
